## Conventions

- **Bold** for things you click or type.
- `code font` for parameter names, net names, file paths, and CLI commands.
- Blockquotes (`>`) hold supplementary notes.
- Screenshot references (`![caption](screenshots/foo.png)`) mark images
  that will be added later.
